How to Set up and Use ChatGPT in Linux Terminal

Artificial Intelligence has taken center stage as the tech sector's new darling. Platforms like ChatGPT and its kin are thriving online, streamlining a plethora of routine tasks for users. In our previous releases, we've covered the basics of crafting a ChatGPT-esque chat-bot, leveraging Bing AI across various browsers, among other topics. But what if you're operating within the Linux ecosystem and yearn to integrate the savvy of AI chat-bots into your workflow? That's where ShellGPT enters the scene, delivering ChatGPT's capabilities straight to your Linux Terminal. ShellGPT is adept at offering insightful responses, crafting intelligent suggestions, and even scripting commands and code, enhancing your command line mastery. This guide is set to demonstrate the installation of the ShellGPT command line utility and introduce the utilization of a ChatGPT-mimicking AI within the Linux command line for a transformative user experience.

Install Python

Python stands as a cornerstone in the toolkit of modern computing, and it's the foundation upon which ShellGPT—a command line incarnation of ChatGPT—is constructed. Indeed, Python, in concert with other programming languages, forms the backbone of ChatGPT's architecture. Most contemporary Linux distributions come with Python already in place. To begin, ascertain the version of Python your Linux machine is equipped with by executing the following command:

python3 --version

Install Pip Package Manager 

sudo apt-get -y install python3-pip
pip3 --version

Install Venv Module 

sudo apt install python3-venv

Set up the Environment 

mkdir <your_directory_name>
cd <new_directory_name>
python3 -m venv <virtual_environment_name>
source <virtual_environment_name>/bin/activate
(<virtual_environment_name>)<username>@<system_name>

Get Your OpenAI API Key 

export OPENAI_API_KEY=<your_OpenAI_API_key_here>
env #verify the enviroment variable and make sure the API key is correct.

To make the API key permanent open the .bashrc file and add the API key at the end of the file.

export OPENAI_API_KEY=<your_OpenAI_API_key_here>
source .bashrc

Install ShellGPT to Use ChatGPT 

pip3 install shell-gpt

Use ShellGPT for Queries 

sgpt "<your_query>"
sgpt "mass of sun"

ShellGPT transforms your Linux terminal by infusing it with the intelligence of ChatGPT. This tool simplifies command line interactions, catering to novices and experts alike, while also offering a suite of extra functionalities. As highlighted earlier, ShellGPT is crafted to evolve and adapt through user interaction, becoming increasingly adept over time. However, it's crucial to exercise caution and avoid disclosing any confidential information or proprietary company code to any AI system.
